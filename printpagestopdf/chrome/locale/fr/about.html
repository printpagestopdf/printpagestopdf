Print Pages to Pdf

  • À propos du module
  • Contributions
Ce module vous permet d' imprimer le contenu d'un ou plusieurs onglets, de marque-pages et de Scrapbook(Plus) s'il est installé, dans un document PDF (et bien sûr avec l'imprimante à partir de toute visionneuse PDF).

Le processus se fait entièrement au niveau local, aucune donnée n'est transmise sur Internet. La bibliothèque open source wkhtmltopdf est utilisé pour la génération de PDF. Malgré la maturité et les bon résultats dont à fait preuve cette bibliothèque, le module ne fournit que les résultats produits par wkhtmltopdf.

Contributions

wkhtmltopdf de Jakob Truelsen
La bibliothèque open source avec laquelle la conversion des pages au format Pdf est effectuée. Plus d'informations sur le projet sur wkhtmltopdf ou le site Internet de Jakob Truelsens.

Scrapbook(Plus) de Gomita et haselnuss
Les deux extensions Scrapbook de Gomita et celle qui en est dérivée Scrapbook Plus de haselnuss ont fourni des morceaux de code à ce module. L'utilisation de ces extensions demeure encouragée.

Screengrab de Andy M
Le code source du module Screengrab de Andy M a été utilisé pour permettre de générer une image à partir d'un site Internet (au format pdf).
