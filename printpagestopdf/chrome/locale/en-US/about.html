Print Pages to Pdf

  • About this AddOn
  • Contributors
This AddOn gives you the ability, to print the content of one or more Browsertabs, Bookmars and, if installed Scrapbook(Plus) into one Pdf Document  (and then to print it with your favourite Pdf Viewer).

The AddOn works fully locally, that means, no Data will be sent to the internet. This is done by using the open Source  Library  wkhtmltopdf. This library gives excellent results in converting Webpages to pdf; but  nevertheless, the AddOn is limited to the capabilities of  wkhtmltopdf.

Contributors

wkhtmltopdf from Jakob Truelsen
This is the open Source library that does the conversion from Html to Pdf. More about the project at wkhtmltopdf or Jakob Truelsens Page.

AddOn Scrapbook(Plus) from Gomita and haselnuss
Print Pages to Pdf is using part of the code from the  Addon Scrapbook from Gomita and the based on this  Scrapbook Plus AddOn from  haselnuss Both are excellent AddOns where you should have a look on.
